How to Use

1. Select the type of conversion you want to perform from the "Conversion Type" dropdown.
2. Choose the units you want to convert from and to using the respective dropdowns.
3. Enter the value you wish to convert in the "From Unit" input box.
4. Click the "Convert" button to see the result in the "To Unit" box.
5. Use the "Swap Units" button to quickly switch the from and to units.
6. Click "Clear" to reset all fields and start a new conversion.

Privacy and Security

All conversions are done locally in your browser. No data is sent to any server, ensuring complete privacy and security.
